# Play Music 🎵

**Play Music** es una aplicación de reproducción de música moderna y elegante construida con Flutter. Ofrece una experiencia de usuario fluida con un diseño oscuro minimalista, gestión de listas de reproducción y reproducción en segundo plano.

## ✨ Características

- 🎶 **Reproducción de Música Local:** Escanea y reproduce archivos de audio almacenados en tu dispositivo.
- 📂 **Gestión de Playlists:** Crea, edita y organiza tus propias listas de reproducción personalizadas.
- 🕒 **Historial Reciente:** Accede rápidamente a las últimas canciones que has escuchado.
- 📱 **Reproducción en Segundo Plano:** Sigue escuchando tu música mientras usas otras aplicaciones, con controles integrados en la barra de notificaciones.
- 🎨 **Interfaz Moderna:** Diseño oscuro (Dark Mode) con efectos de desenfoque (Glassmorphism) y transiciones suaves.
- 🖐️ **Gestos Intuitivos:** Desliza para cambiar de canción o cerrar el reproductor.
- 📳 **Feedback Háptico:** Vibración integrada para una mejor experiencia táctil.

## 🛠️ Tecnologías Utilizadas

- **Flutter & Dart:** El núcleo de la aplicación.
- **just_audio & just_audio_background:** Para una reproducción de audio robusta y soporte de notificaciones.
- **on_audio_query:** Para escanear y organizar la biblioteca musical del dispositivo.
- **shared_preferences:** Para persistencia de datos (playlists e historial).
- **permission_handler:** Gestión eficiente de permisos de almacenamiento.
- **audio_video_progress_bar:** Barra de progreso interactiva y estética.

## 🚀 Instalación y Uso

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/franciscorojas27/Play-Music.git
   ```
2. **Instalar dependencias:**
   ```bash
   flutter pub get
   ```
3. **Configurar Iconos de Lanzamiento:**
   ```bash
   flutter pub run flutter_launcher_icons
   ```
4. **Ejecutar la aplicación:**
   ```bash
   flutter run
   ```
